# VSCode Great Icons

A big pack of icons (100+) for your files.

![preview](https://images.emmanuelbeziat.com/settings.json_-_demo_-_Visual_Studio_Code-ur2g8-ldpqc.jpg)

I can add more if you need, [open a new issue](https://github.com/EmmanuelBeziat/vscode-great-icons/issues) to ask which extension you want.

Enjoy ☺

## How to use

After installation and activation, you should go in settings (`file` → `Settings` on Windows, or `Code` → `Settings` on OSX), choose `File Icon Theme`, and select `VSCode Great Icons`.

## Sources

Most of these icons come from
